## Getting the Code

```sh
  git clone https://github.com/oscarmarina/XstateController.git
  cd XstateController
```

### Installation

```js
npm i && npm start
```

### Serve with `vite`

```js
  npm start
```

### Tests with web test runner`

```js
  npm run test
  // or
  npm run test:watch
```
